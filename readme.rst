Read Naming Format
==================

RNF specification
-----------------

TeX source code of the RNF specification is in file `RNF.tex`_. XeLaTeX is the preferred way of compilation:

.. code:: bash

    xelatex RNF.tex

It can be compiled also with pdfLaTeX:

.. code:: bash

   pdflatex RNF.tex
